flask-bones
===========

I've been reusing this pattern for Flask applications and decided to stop repeating myself.

## Development

```bash
$ python run.py
```

## Deploying on Heroku

1. Ensure you have the [heroku-toolbelt](https://toolbelt.heroku.com/) installed.

```bash
$ heroku create
$ git push heroku master
$ heroku open
```
